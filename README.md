### Add Your Logo (2 min)
1. Convert your PDF to PNG/SVG using [CloudConvert](https://cloudconvert.com/pdf-to-svg)
2. Save as `assets/images/logo.svg` or `logo.png`
3. Refresh page - logo appears!

### Configure Form (5 min)
1. Get free key from [web3forms.com](https://web3forms.com)
2. Edit `js/main.js` line ~180
3. Replace `YOUR_WEB3FORMS_ACCESS_KEY` with your key

### Deploy (2 min)
1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag `landing` folder
3. Get live URL!
