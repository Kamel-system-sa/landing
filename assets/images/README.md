# Images Folder

Place your logo and images here.

## Recommended Images

1. **logo.png** or **logo.svg**
   - Dimensions: 150x40px (or similar aspect ratio)
   - Format: PNG with transparent background or SVG
   - Usage: Navigation bar logo

2. **hero-image.png**
   - Dimensions: 800x600px minimum
   - Format: PNG or JPG
   - Usage: Hero section illustration (optional)

3. **feature-icons/** (if using custom icons)
   - Dimensions: 64x64px
   - Format: SVG or PNG
   - Usage: Feature section icons

4. **testimonial-avatars/** (if using real photos)
   - Dimensions: 100x100px
   - Format: JPG or PNG
   - Shape: Square (will be displayed as circle)

## Image Optimization Tips

- Compress images before uploading (use TinyPNG.com)
- Use WebP format for better compression (with fallbacks)
- Keep images under 200KB each for fast loading
- Use descriptive filenames (e.g., `kamel-logo.png` not `image1.png`)
